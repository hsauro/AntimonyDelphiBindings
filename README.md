# Antimony Delphi Bindings
This project includes Delphi bindings to libAntimony (SBML), as wrapper class and an example of how to use it.

AntimonyAPI.pas is the bindings file itself
AntimonyWrapper.pas is the wrapper file
The project file shows a long exmaple of how to use the wrapper.

You can get teh antimony dlls and windows runtime from 

https://github.com/sys-bio/antimony

Got to releases and download antimony-windows-latest-release.zip
