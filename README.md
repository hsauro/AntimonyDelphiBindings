# Antimony Delphi Bindings
This project includes Delphi bindings to libAntimony (SBML), a wrapper class, and an example of how to use the wrapper.

AntimonyAPI.pas is the bindings file itself

AntimonyWrapper.pas is the wrapper file

The project file shows an example of how to use the wrapper.

You can get the antimony dlls and windows runtime files from 

https://github.com/sys-bio/antimony

Go to releases and download antimony-windows-latest-release.zip

Copy libantimony.lib in the same folder as the executable as well as teh microsoft (140) runtime libraries
